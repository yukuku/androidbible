When you run this application, you will immediately see the Bible text. Press the button below that is showing the verse address, like Genesis 1, to go to other parts of the Bible. Press the left and right button to move to the previous and next chapters.

Note: Some of the verses, especially in the book of Psalms, has numberings that differ between different translations and other translations (e.g. KJV). The chapter and verse numbers of the specific translation are written in parentheses.

Moving to Other Parts of the Bible

To move to another part of the Bible, you can press the verse address button, like Genesis 1

Hold (long-press) the verse address button to open the list of recent verses.

On the Direct tab, you can directly type the verse address. How to write the verse address is: Write the first 3 or so characters of the book name, then the chapter number, then a space or a colon, then the verse number. For example Luk 3:11 or 1Pe 2:5 (1 Peter 2:5).

However, for your convenience, you can also write it in the following ways:

  • Type just the chapter number to go to the chapter on the same book. Example: 8
  • Type the chapter number followed by the verse number. Example: 3 16, 12:4
  • Type the book name just enough. Example: songs (Song of Solomon), 1k (1 Kings), ps (Psalms)
  • Concatenate the book name and the chapter number: Joh3, Gen48

Cross Reference Verse

In Bahasa Indonesia Terjemahan Baru version, in some verses, it provides cross reference to other related verses. The verse with cross reference provided is displayed with ※. For example Genesis 1 verse 3:

3 Berfirmanlah Allah: "Jadilah terang." Lalu terang itu jadi. ※

If this symbol ※ is tapped, cross reference verses will be displayed.

Markers

Press MENU and select Markers to show all markers that you have created. There are 3 types of marker to be listed:

  1. Bookmarks. With bookmarks, you can mark verses that you think are important. You can also give a title and labels to each bookmark. A label can be made by clicking Add... when you create a bookmark and then select Create label. Label is useful if you want to put your bookmarks into categories.
  2. Notes. This feature helps you to put a note related to the verse where it attached.
  3. Highlights. It provides background color to the verses that you found them interesting so they will look different from other verses.

If you have created labels attached to bookmarks, all labels will be displayed in this screen. Press one of the bookmarks to open it, or hold to change the title, labels, or to delete it.

By pressing MENU, you can also Backup or Restore. Use this to ensure that your bookmaks, notes, and highlights are stored properly in SD Card or USB Storage as a backup in case there is a problem with your device. Use Transfer to create a backup and send it to another application. This could be useful to move application data to other devices. Press Export to send markers via email, Dropbox, bluetooth, etc (depends on apps you have installed in your device) in html format to other device. It is not for backup purpose, but to help you summarize all your markers in readable format.

Search

You can find verses that contain certain words. Press the search button (with the magnifying glass symbol).

When the search screen opens, type the words that want to look for and click the button. The verses that contain all the words you entered will be displayed. For example, if you type god son, then even Genesis 4:25 would be displayed because it contains verses the word "god" and "son", although not contiguous. If you type just, you will also find "justice", "justify", etc.

  • Use quotes to search for exact phrases. For example, "our father".
  • Use plus sign (+) exactly before a word to search a whole word. For example, +eve will find the exact word "eve" or "Eve" but not "eventually" or "evening".

Devotion

This feature is only available in the Indonesian version of Bible for Android (search for "Alkitab" in Android Market).

Press MENU and select Devotion to read some reflections which are provided every day. When you first open the devotion, you must be connected to Internet, because the devotion materials are available online. However, when you connect, the devotion from that day to 30 days later will be downloaded and stored so you do not need to be connected to the Internet to read the devotion of the following days.

Press Change date (with calendar symbol) to read devotions from different sources. Press the left arrow or right arrow buttons to move between days. Open the menu to copy the contents of the devotion or to share them with others.

Some devotions contain verse addresses that you can press to jump directly to the verses.

You can set a reminder by pressing MENU and select Reminder. This menu need another modul to be downloaded from Google Play. If you already downloaded the module before, it will not download the module again but will instantly display alarm settings.

Progress

Press MENU and select View Progress to get all progresses displayed. Progress is designed to mark last verse that you read. There are 5 built in progresses. Select a progress in this screen to go to the verse where the progress is saved. Press and hold for a second to edit progress name or to delete progress.

Songs

This feature is only available in the Indonesian version of Bible for Android (search for "Alkitab" in Android Market).

Press MENU and select Songs to read Christian songs.

Press Search to open the song list. To get the newest song data, erase the old data by opening the menu at this screen.

Version

You can read another version of the Bible (in different languages or different translations) by selecting the Version menu (with 1<->2 framed in 2 boxes symbol). At the beginning, there will be only one version available. Tap on Other versions... to add and manage Bible versions.

In the managing versions screen, there will be some pre-set versions that are ready to be downloaded. You will need to download the data file for that version beforehand. The data file will be stored in the SD Card. Each version occupies around 5 MB of space. After you download them once, you don't need to download them again.

If you have PDB files in PalmBible+ format, those files can be read by this application. This application does not directly read .pdb files, instead it will convert the .pdb files to its own format whose file name ends with .yes beforehand. This file is stored on the SD Card or USB Storage in /bible/yes folder with the file name generated depending on the PDB file opened. Long-press on one of the items on the version list, and select Details to check the location of the files and so on.

The checkboxes on the right of the versions indicates whether the versions are shown on the version selection dialog.

Verse Menu

Tap a verse so that the verse is selected, and then some menus will appear.

  • Copy or share (with document and links symbol) has 2 menus: Copy verse to copy the selected verses to the clipboard and to be pasted into other applications. Share to send the selected verse by email, bluetooth, Twitter, Facebook, etc. (Depending on the application installed on your device.)
  • Add marker (with + symbol) has 3 menus: Add bookmark, Write note and Highlight, to add bookmark, note or highlight respectively.
  • Save progress (with location pin symbol) to save progress on the selected verse

If you select more than one verse, Save progress, Add bookmark and Write note menus will not appear since these features can only be applied to single verse only.

If your device has NFC (Near Field Communication) capabilities, you can show the same Bible verse you're reading on your friend's device, which need to have NFC capability as well.

Touch the back of your device to the back of your friend's device, and you'll see the outer space view on the display. Tap the screen to send the verse address to your friend. Your friend's device will perform one of these:

  • Jump to the verse you send, if the Bible application is already open.
  • Open the Bible app and go to the verse that you send, if the app is not open yet.
  • Open the Bible app page on Google Play so that your friend can download and install the app.

Frequently Asked Questions

How do I get the NIV, ESV, NKJV, or other translations of the Bible?

We did not include these versions because they are not free (as in liberty) or free (as in gratis), so we can not include it in this application. You can read it in, for example, www.biblegateway.com for those versions if you want to consult them.

How do I search for a word like "word" without getting results containing other terms such as "sword" or "words"?

In order to find an exact word without affixes or other words that are not expected, add a plus sign (+) immediately before the word. For example: +word.

Is this app also available for iOS or Windows or any other platforms?

No, this app is only available for Android. However, the source code is available at https://github.com/yukuku/androidbible if you want to port it to other platforms.
